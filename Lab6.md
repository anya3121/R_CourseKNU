6. Знайти кількість значень < 0 для кожного стовпця. Використати свою функцію.
```
function1 <- function(mat) {
  col <- ncol(mat)
  for (i in 1:col) {
    col[i] <- sum(mat[,i]<0)
  }
  col
}
function1(mat)
```
```
[1] 3 5 3 6 3
```
